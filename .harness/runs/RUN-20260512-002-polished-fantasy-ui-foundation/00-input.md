# Input

User requested upgrading the existing playable Dice Monsters-style tactical game UI into a polished 2D fantasy strategy game screen.

Key requirements:

- Ornate fantasy HUD, dark stone dungeon board, blue vs red player sides.
- Glowing Heart/Core bases.
- 2D monster tokens/standees with HP/level/selection states.
- Animated dice rolling, summon, movement, attack, damage, core hit, phase banner, crest counter pop.
- Preserve reducer/rule source of truth and existing gameplay actions.
- No official Yu-Gi-Oh! assets/logos/names/copied artwork.
- Verify with `npm run build`, and runtime/manual smoke where possible.
