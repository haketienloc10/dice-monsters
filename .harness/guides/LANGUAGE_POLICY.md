# Language Policy

Generated Harness artifacts nên dùng ngôn ngữ người dùng đang dùng trong task hiện tại.

Giữ nguyên dạng gốc cho:

- technical identifiers;
- code;
- commands;
- file paths;
- config keys;
- error messages;
- logs;
- API fields;
- schema keys;
- package names;
- copied tool output.

Nếu user yêu cầu một ngôn ngữ cụ thể, dùng ngôn ngữ đó cho response và artifact liên quan.
