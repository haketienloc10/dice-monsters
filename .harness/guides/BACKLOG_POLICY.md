# Backlog Policy

`.harness/backlog/HARNESS_BACKLOG.md` chứa proposal cải tiến Harness local cho target repository.

## Khi thêm proposal

Thêm proposal khi một run phát hiện rule, template, script, hoặc validation improvement có thể tái sử dụng.

Không sửa guide/template/script ngay nếu task hiện tại không yêu cầu. Ghi proposal trước để người duy trì target repo quyết định.

## Installer rule

Installer không overwrite `.harness/backlog/HARNESS_BACKLOG.md` nếu target đã có file này.
