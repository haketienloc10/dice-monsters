# Project Glossary

Last reviewed: 2026-05-12

- `Board`: 13 by 9 dungeon grid used for cores, dungeon tiles, and monsters.
- `Core`: player base tile with HP; damaging the opponent core can end the game.
- `Crest`: resource generated by dice faces, including summon, move, attack, defense, magic, and trap.
- `DiceDefinition`: static dice config with level, monster association, faces, and tile shape.
- `GameAction`: typed reducer action dispatched by UI or AI.
- `GameEvent`: latest reducer event used by UI for animation and status feedback.
- `GamePhase`: one of `roll`, `summon`, `action`, or `gameOver`.
- `MonsterDefinition`: static monster stats and optional skill text.
- `MonsterInstance`: a summoned monster on the board with owner, position, HP, and action state.
- `P1`: human-controlled Blue Warden in the current initial state.
- `P2`: AI-controlled Crimson Overlord in the current initial state.
- `TileShape`: dungeon tile shape offsets used during summon placement.
